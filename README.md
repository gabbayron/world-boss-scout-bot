
World Boss Scout Bot

Simple Discord bot to track scouting for WoW world bosses.

Commands:
/setup-board channel:#channel
/scout boss:<boss> layer:<layer>
/scout-remove
/boss-dead boss:<boss>
/refresh-board

Setup:

1. Install Node LTS
2. npm install
3. Copy .env.example to .env and fill values
4. npm run deploy:commands
5. npm run dev
